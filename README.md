# React Playground

A collection of small React apps built with Vite and TypeScript.

## Projects

| Project | Description |
|---------|-------------|
| **react-counter** | Counter app |
| **react-background-changer** | Background color changer (Tailwind) |
| **react-password-generator** | Password generator |
| **react-website-clone** | Website clone |

## Running a project

Each project runs independently. From a project folder:

```bash
cd <project-name>
npm install
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`).
